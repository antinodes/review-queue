Paste a GitHub personal access token with repo scope:

This app calls the GitHub API to fetch your PR review queue. The token never leaves your browser — it's stored in localStorage and sent directly to api.github.com.

Option 1: Create a personal access token with repo scope. Fine-grained tokens let you restrict to specific repos.

Option 2: If you use the GitHub CLI, run gh auth token and paste the output. This token may rotate when you re-authenticate.

Stored in localStorage on this device only.
